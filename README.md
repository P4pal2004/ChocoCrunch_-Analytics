# ChocoCrunch_-Analytics
Streamlit + SQL + ETL +ED A +feature engineering +data cleaning + API data extration for chococrunch analytics
Project Overview

ChocoCrunch Analytics is a data analysis project focused on exploring nutritional information of chocolate products. Using Python and SQL, this project extracts, cleans, and visualizes chocolate data to uncover insights such as calorie content, sugar levels, and product distribution.

Features

✅ Extract data from CSV/Database using Python

✅ Data cleaning and preprocessing

✅ Data visualization using Matplotlib, Seaborn, and Plotly

✅ Insights on calories, sugar content, and product categories

✅ SQL queries for deeper data analysis

✅ Streamlit dashboard for interactive exploration

Tools & Technologies

Programming Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn, Plotly, Streamlit, PyMySQL

Database: Sqllite3

Visualization: Charts, Scatter Plots, Box Plots, Histograms

Version Control: Git & GitHub

Installation

Clone the repository:

git clone https://github.com/yourusername/ChocoCrunch-Analytics.git


Navigate to the project folder:

cd ChocoCrunch-Analytics


Install required Python libraries:

pip install -r requirements.txt


Run the Jupyter Notebook for analysis:

jupyter notebook


(Optional) Run Streamlit dashboard:

streamlit run app.py
